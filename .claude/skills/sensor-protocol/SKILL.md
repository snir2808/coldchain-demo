# פרוטוקול החיישנים

כל מכשיר מעקב משדר הודעה בינארית באורך קבוע של 24 בתים.

## מבנה ההודעה

| בתים | שדה | טיפוס | הערות |
|---|---|---|---|
| 0-3 | מזהה מכשיר | uint32, big-endian | מוצג כמחרוזת הקסה, למשל A107C344 |
| 4 | טמפרטורה | int8, צלזיוס | חתום! מטעני הקפאה משדרים ערכים שליליים |
| 5 | דגלים | bitfield | ראה פירוט למטה |
| 6-7 | סוללה | uint16, little-endian | ערך ADC גולמי, מילי-וולט = ערך כפול 4 |
| 8-11 | מונה רצף | uint32, little-endian | עולה בכל שידור |
| 12-22 | שמור | | אפסים בגרסה הנוכחית |
| 23 | CRC-8 | poly 0x07, init 0x00 | על בתים 0-22 |

## דגלים (בית 5)

| ביט | משמעות |
|---|---|
| 0 | תנועה זוהתה |
| 1 | דלת פתוחה |
| 2 | חבלה |
| 3 | מקור מתח חיצוני |
| 4 | סוללה תקינה |
| 5-7 | שמור |

## מלכודות ידועות

- הטמפרטורה היא int8 חתום. קריאה כ-uint8 הופכת מינוס 18 ל-238.
- מזהה המכשיר הוא big-endian, שאר השדות הרב-בתיים little-endian.
- הודעה עם CRC שגוי נזרקת, לא מתקנים ולא מנחשים.
